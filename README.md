# Namma Vastra

Namma Vastra is a modern Android e-commerce application developed for traditional fashion and saree marketplace management. The application provides a premium digital shopping experience with dedicated customer and seller modules, real-time inventory synchronization, secure authentication, and a modern user interface built using Jetpack Compose.

## Features

* User Authentication using Firebase Authentication
* Seller and Customer Role-Based Access
* Product Upload and Management
* Real-Time Firestore Database Integration
* Shopping Cart and Cart Validation
* Modern UI using Jetpack Compose
* MVVM Architecture
* State Management using StateFlow
* Responsive Navigation System
* Firebase Cloud Backend Integration

## Tech Stack

* Kotlin
* Jetpack Compose
* MVVM Architecture
* Firebase Authentication
* Firebase Firestore
* Navigation Compose
* StateFlow
* Coroutines
* Coil Image Loading

## Installation

1. Clone the repository:

```bash
git clone https://github.com/abhinavas-10/NammaVastra.git
```

2. Open the project in Android Studio.

3. Sync Gradle dependencies.

4. Connect Firebase project configuration.

5. Run the application on emulator or Android device.
